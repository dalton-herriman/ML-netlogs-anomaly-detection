# Security Policy

## Supported Versions

This project is a portfolio / reference implementation. Only the `main` branch receives security updates.

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please **do not** open a public GitHub issue.

Instead, report it privately via one of the following channels:

- **Email:** daltonclayherriman@gmail.com
- **GitHub Security Advisories:** <https://github.com/dalton-herriman/ML-netlogs-anomaly-detection/security/advisories/new>

Please include:

- A description of the issue and its potential impact.
- Steps to reproduce (a minimal proof-of-concept is ideal).
- Any suggested remediation or workaround.

## Response Expectations

- Acknowledgement of receipt within **3 business days**.
- Triage and severity assessment within **7 business days**.
- A target fix window will be communicated after triage.

Please give us a reasonable window to remediate before any public disclosure. We will credit reporters in the release notes unless anonymity is requested.
